# Projet-de-Structuration-de-Produits-D-riv-s
L'objectif est de déterminer le coupon d'équilibre (*Fair Value*) via une méthode de Monte Carlo sous hypothèse de volatilité constante (Black-Scholes), puis d'analyser la sensibilité du produit aux paramètres de marché (Grecs) ainsi que sa performance historique.
